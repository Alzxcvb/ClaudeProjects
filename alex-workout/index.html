The Program

6-Day Push/Pull/Legs — Built for Mass

Weekly Schedule

Day Session Focus
Monday Push A Chest, Shoulders, Triceps + Abs
Tuesday Pull A Back Width, Biceps
Wednesday Legs A Quads, Hamstrings, Calves + Core
Thursday Push B Chest, Shoulders, Triceps
Friday Pull B Back, Biceps + Abs
Saturday Legs B Quads, Hamstrings, Calves + Core
Sunday Rest Walk, stretch, sauna

Why PPL?

Each muscle group gets hit twice per week — the sweet spot for beginners gaining mass. Directly targets shoulder width, chest, and back (the V-taper). Six days matches your availability; Sunday is full rest.

Progression Scheme

Phase 1 — Form Foundation (Weeks 1-3)

  • Drop to 3 sets per exercise, lighter weight (50-60% effort)
  • Focus: learn the movement, nail the form, build the habit
  • Add 2.5kg to barbell lifts each week if form is solid

Phase 2 — Volume Building (Weeks 4-8)

  • Full sets as programmed
  • Add 2.5kg per week on compounds (bench, OHP, squat, row)
  • Add 1 rep per week on accessories (laterals, curls, etc.)

Phase 3 — Intensification (Weeks 9-12)

  • Progressive overload tracking
  • When you can't add weight, add a set (up to 5 sets)
  • Week 12 = Deload week (2 sets, 60% weight)
